# Contributing to Evently

Thanks for your interest in contributing! Here's how to get started.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push to the branch (`git push origin feature/my-feature`)
5. Open a Pull Request

## Development Setup

1. Install [.NET 10 SDK](https://dotnet.microsoft.com/download)
2. Install [Node.js](https://nodejs.org/) (for Tailwind CSS)
3. Clone the repo and run `dotnet run`

## Guidelines

- Keep changes focused and minimal
- Follow existing code style and C# conventions
- Add or update tests for new functionality
- Update documentation if needed

## Reporting Issues

Open an issue on GitHub with:

- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior

## Code of Conduct

Be respectful and constructive. We are all here to learn and build together.
